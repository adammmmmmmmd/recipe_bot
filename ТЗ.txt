Бот для анализа питания

Создайте телеграм бота, который предоставляет рецепты и анализирует питание.

Поиск рецептов:

Пользователь может запросить рецепты по названию блюда или ингредиентам .
Бот предоставляет подробные рецепты, включая список ингредиентов, шаги приготовления и калорийность блюда.
Подсказка: Используйте сторонний API для поиска рецептов

Калорийный анализ:

Пользователь может отправить боту список употребленных продуктов за день.
Бот производит анализ и возвращает общую калорийность, белки, жиры и углеводы потребленных продуктов.
Используйте сторонний API для анализа калорий. 

Интерактивные тренировки:

Пользователь может запросить бота советы по физической активности и получить краткие интерактивные тренировки для поддержания активного образа жизни.
Задание со звездочкой.

Личный кабинет:

Пользователь может создать свой личный кабинет, где хранится история запросов, сохраненные рецепты и статистика потребления питательных веществ.
Для личного кабинета используйте базу данных для хранения информации о пользователях, их запросах и сохраненных рецептах.

Напоминания:

Бот может отправлять уведомления с напоминаниями о времени приема пищи или предложениями новых рецептов.
Настройте напоминания через schedule или аналогичную библиотеку.
